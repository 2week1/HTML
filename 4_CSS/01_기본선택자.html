CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬 수 있다.

모든 (전체) 선택자 : *

현재 이 문서 상에 모든 요소들을 다 선택하고자 할 때 사용

        * { 
            스타일 속성 : 값;
            스타일 속성 : 값;
        }

태그 선택자 : 태그명

현재 이 문서 상의 해당 태그를 모두 선택하고자 할 때 사용

        태그명 {
            스타일속성 : 값;
        }

아이디 선택자 : #아이디명

현재 이 문서 상의 특정 html 요소 "하나"만을 선택하고자 할 때 사용
단, 해당 요소에서 id 속성을 이용해서 고유한 아이디를 부여해야 함 

        #아이디명 {
            스타일속성 : 값;
        }

  4. 아이템4

클래스 선택자 : .클래스명

현재 문서 상에 특정 html요소 "여러 개"를 선택하고자 할 때 사용 

        .클래스명 {
            스타일속성 : 값;
        }
